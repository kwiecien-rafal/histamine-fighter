# Histamine Fighter

Agentic AI meal suggestions for people managing histamine intolerance.

> **Status:** Work in progress. Active development — APIs, schema, and UI are unstable.

## Stack

- **Backend:** FastAPI, SQLAlchemy (async), Pydantic v2
- **Frontend:** React, Vite, TypeScript, Tailwind
- **LLM:** Swappable provider (Modal, OpenAI, Anthropic, Gemini, Ollama)
